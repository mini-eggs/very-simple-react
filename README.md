# very-simple-react
Showing how to extend components/how easy it is

## run in dev

1) yarn install  
2) composer install  
3) open two terminals and cd to this location  
4) in terminal one run `gulp watch`  
5) in terminal two run 'npm run server'  
6) navigate to http://localhost:3000/
